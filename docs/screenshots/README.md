# Screenshot Checklist

Use this folder for portfolio-ready Streamlit screenshots referenced by the main README.

Recommended captures:
- `command-center.png`
- `flow.png`
- `handoffs-quality.png`
- `enablement.png`

Capture guidelines:
- use desktop width
- keep the Streamlit sidebar visible
- show the global filter bar and KPI strip
- prefer one clean hero screenshot per workstream rather than many repetitive captures
