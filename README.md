# IncidentOps PMI Hub

A process-mining insights hub for Incident Management event logs. Built to demonstrate Python, SQL, BABOK FRD practices, and BPMN (As-Is / To-Be).

## What this will include
- Postgres analytics schema + views (process metrics, variants, bottlenecks, handoffs)
- Streamlit multi-stakeholder dashboard
- Portfolio artifacts: FRD, KPI dictionary, traceability, BPMN diagrams

## Tech
- Python (ETL + app)
- Postgres (hosted, provider TBD)
- Streamlit + Plotly
